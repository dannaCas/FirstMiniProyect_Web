# FirstMiniProyect_Web
    Sistema Web de intercambio de Regalos 

# Básicos para el GitHub
### Primera vez - Clonar el proyecto
```bash
git clone https://github.com/dannaCas/FirstMiniProyect_Web.git
```

### ANTES de empezar a trabajar (HAZLO SIEMPRE) 
```bash
# Actualizar dev
git checkout dev
git pull origin dev
```

# Ver qué cambió
git status

# Agregar TODOS los cambios
git add .

# O agregar archivos específicos
git add archivo1.js archivo2.css

# Hacer commit con mensaje descriptivo
git commit -m "Añadí formulario de login con validación"

# Subir cambios a GitHub
git push origin dev
